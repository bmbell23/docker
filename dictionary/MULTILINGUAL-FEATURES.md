## 🔄 Translation Tool

Full sentence translation between any supported language pair:

**How to use:**
1. Click the **Translate** tab
2. Select **From** language (default: English)
3. Select **To** language (default: Spanish)
4. Type or paste your text
5. Click **Translate**
6. See the translation appear instantly

**Examples:**
- English → Spanish: "Hello, how are you?" → "Hola, ¿cómo estás?"
- Spanish → English: "Buenos días" → "Good morning"
- English → French: "Thank you" → "Merci"
- English → Mandarin: "Hello" → "你好"

## 🔧 Technical Details

### APIs Used
- **Dictionary API**: Free Dictionary API (dictionaryapi.dev)
- **Translation API**: MyMemory Translation API (mymemory.translated.net)
- **Autocomplete**: Datamuse API (datamuse.com)

### Limitations
- Translation API is free tier (may have occasional inaccuracies)
- Translations are word-level, context may vary
- Some rare words may not have all translations
- Pronunciation guides coming in future update
