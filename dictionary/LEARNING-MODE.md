## ⭐ FAVORITES SYSTEM

### How to Star Words

1. Look up any word in the Dictionary tab
2. Click the **star icon (☆)** next to the word title
3. Star turns gold (★) when added to favorites
4. Click again to remove from favorites

### Viewing Your Favorites

1. Click the **⭐ Favorites** tab
2. See all your starred words sorted by date (newest first)
3. Each favorite shows:
   - **Word** (clickable to look up again)
   - **When you starred it** (e.g., "today", "2 days ago")
   - **Part of speech** and **definition**
   - **Remove button** to unstar

### Features

- **Persistent storage** - Favorites saved in browser (localStorage)
- **Date tracking** - See when you starred each word
- **Quick lookup** - Click any favorite to see full definition
- **Easy management** - Remove button on each card

## 🎓 LEARNING MODE

### Overview

A complete language learning system with:
- **9 languages** to learn

- **Streak tracking** to keep you motivated
- **Common vocabulary** (30+ essential words)

- 🇨🇳 Mandarin Chinese
- 🇯🇵 Japanese
- 🇰🇷 Korean
- 🇩🇪 German
- 🇷🇺 Russian
- 🇵🇱 Polish

### How to Use Learning Mode

#### 1. Choose a Language

1. Click the **🎓 Learn** tab
2. See all 9 languages with progress bars
3. Click any language card to start learning

#### 2. Take Quizzes

**Quiz Format:**
- Question: "What is [English word] in [Language]?"
- 4 multiple choice options
- Click your answer
- Instant feedback (green = correct, red = wrong)
- Correct answer always shown

**Example:**
```
What is "hello" in Spanish?

hello

[ ] Adiós
[ ] Por favor
[ ] Hola      ← Correct!
[ ] Gracias
```

#### 3. Track Your Progress

**Stats Displayed:**
- **Correct** - Number of correct answers
- **Total** - Total questions answered
- **Streak** - Current correct answer streak

**Progress Bar:**
- Shows percentage correct for each language
- Updates in real-time
- Saved permanently in browser

### Vocabulary Covered

**30 Common Words:**
- Greetings: hello, goodbye, please, thank you
- Basic: yes, no, water, food, house
- People: family, friend, person
- Emotions: happy, sad, love
- Descriptors: good, bad, big, small
- Numbers: one, two, three
- Time: today, tomorrow, yesterday, morning, night, day, time
- Places: place

### Learning Tips

1. **Start with one language** - Focus on mastering one before moving to another
2. **Practice daily** - Even 5 minutes helps build vocabulary
3. **Use favorites** - Star words you want to practice
4. **Check translations** - Use the Dictionary tab to see full context
5. **Build streaks** - Try to get 10+ correct in a row!

### Progress Tracking

**Saved Data:**
- Correct answers per language
- Total questions per language
- Percentage accuracy
- All data persists in browser

**Example Progress:**
```
Spanish: 45/60 correct (75%)
French: 12/15 correct (80%)
German: 8/20 correct (40%)
```
